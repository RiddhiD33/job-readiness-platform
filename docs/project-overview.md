# Job Readiness Platform

## Project Purpose

The Job Readiness Platform helps students measure and improve their readiness for industry jobs through assessments, projects, internships, profile completion, and career-path-based learning.

The platform focuses on evidence rather than claims.

Instead of relying only on CGPA, skills, or resume statements, students build a readiness score through practical assessments, projects, internships, and profile achievements.

---

## Main Users

### Student

Can:

- Register account
- Complete profile
- Select career path
- Add projects
- Add internships
- Take assessments
- View readiness score
- View assessment history
- Receive recruiter interest notifications
- Manage portfolio links

---

### Recruiter

Can:

- Search students
- Filter students
- View student profiles
- View readiness score breakdown
- Show interest in students
- Request connection with students

---

### Admin

Can:

- Manage career paths
- Manage skill mappings
- Create assessments
- Manage assessment questions
- Review practical tasks
- Manage readiness score rules
- View platform analytics

- Authentication
- Student Profile
- Projects
- Internships
- Assessments
- Readiness Score

Version 1 will NOT include:

- AI Recommendations
- Advanced Analytics
- Automatic Practical Task Evaluation
- Chat System
- Video Interviews
